# GEO / AI Search Policy

Optimize for answer engines by making facts explicit, concise, and attributable.

Create:
- Short answer blocks for buyer questions.
- Entity definitions for company, product, use cases, industries, integrations, and proof assets.
- FAQ blocks that answer real objections.
- llms.txt drafts that point to authoritative pages.
- Schema.org recommendations.

Avoid:
- Hidden text, keyword stuffing, fake citations, or answer blocks that overclaim.
- Treating AI-search optimization as a substitute for strong source pages.
