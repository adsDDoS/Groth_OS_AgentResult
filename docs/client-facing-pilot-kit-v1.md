# GrothOS Pilot Kit v1

Этот пакет используется после квалификации, когда клиент готов запустить первый управляемый пилот текстовой контент-фермы.

Цель пилота: провести один материал через полный контур производства: тема -> черновик -> QA -> выпуск -> подтверждение публикации -> следующий контент-шаг.

### Решение

Выбрать одно:

- Продолжить: контур повторился, можно продлевать пилот или переводить в платный формат.
- Сузить: ценность есть, но нужен один канал/формат/ритм.
- Починить: блокер понятен, сначала чиним процесс.
- Остановить: нет владельцев, источника результата или ценности от контроля производства.

```text

```

### Коммерческий Следующий Шаг

Выбрать одно:

- Продлить пилот на 2 недели.
- Перевести в paid pilot.
- Сузить scope и повторить week 1.
- Запланировать repair sprint.
- Закрыть без продолжения.

```text

```
