- Reuse: переиспользовать идею в другом формате.

- Update: исправить или обновить материал.
- Leave: закрыть без продолжения.

## 6. Week 2 Go / No-Go

Выбрать одно:

- Go: продолжаем тот же контур ещё на 1-2 материала.
- Narrow: оставляем один канал или один формат.
- Repair: сначала исправляем blocker.
- No-go: пилот не продолжаем.

Решение:

```text

why:
required_change:
decision_owner:
```

## Review Is Valid If

- есть факт выхода или понятная причина, почему его нет;
- есть хотя бы один зафиксированный result source;
- blockers названы без размывания ответственности;
- выбран один next content step;
- принято явное week 2 решение.
