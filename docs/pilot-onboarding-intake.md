## Ready To Start

Пилот можно запускать, если заполнены:

- 1 основной канал;
- approval owner;
- QA/release owner;
- result owner;
- weekly cadence;
- первый формат;
- forbidden claims;
- URL/result source.

## Not Ready

Не запускать пилот, если:

- нет человека, который подтверждает публикацию;
- нет канала первой недели;
- никто не может согласовать тему;
- клиент ждёт гарантированный коммерческий результат вместо управляемого производства текста;
- все решения откладываются "после интеграций".
